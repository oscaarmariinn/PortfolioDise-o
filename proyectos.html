Restaurante Dialados

Este proyecto lo realicé para proyecto final del primer curso de 1º de DAW. Este trata de la gestión de reservas de un restaurante totalmente ficticio. En este gaste MYSQL como gestor de la base de datos, que mediante PHP realizaba toda la gestión de las reservas mediante un formulario. Use HTML y CSS básico para la parte visual de la aplicación web.
